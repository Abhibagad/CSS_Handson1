What is CSS?

  1. CSS stands for Cascading Style Sheets.

  2. Css describes how HTML elements are to be display on screen,paper,or in other media.

  3. Css is not neccesary CSS sabves a lot of work. It can control the layout of multiple web pages all at once.

  4. External stylesheets are stored in css files.

CSS can be added to HTML document in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS., is to keep the style s in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS applies a style to multiple element.ID,on the other hand,applies a style to one unique elemrnt. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS,selection are used to target a specific element or range of element on a web page.Once an element has been targeted,a style or set of style can be applied to the element.

There is a wide range of selection available. TWO of the most commonly used are class and ID.Both are used to target element to which a style should be applied.
